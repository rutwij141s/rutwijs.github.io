Projects

Debt Collection System

Technology Stack: Java, Spring Boot, Camunda, Hibernate, MySQL, Postman, Eclipse

Responsibilities:
  • Designed and developed a flexible, Java- and Spring Boot–based workflow-driven debt collection solution leveraging Camunda BPMN 2.0 to automate the end-to-end handling of delinquent accounts.
  • Modeled complex business workflows and decision logic in Camunda, integrating them seamlessly with Spring Boot services and Java components for orchestrating payment reminders, escalations, and legal processes.
  • Developed robust Spring Boot REST APIs for multi-channel communication (SMS, Email, WhatsApp) and integrated them into Camunda service tasks, enabling automated customer outreach and tracking.
  • Implemented real-time payment gateway integration through Java services, automating payment link generation, partial/full payment reconciliation, and updating workflow states in Camunda.
  • Built agent and case management modules using Spring Boot microservices, with workflows such as task assignment, reassignment, and escalation handled via Camunda BPMN models and Java delegates.
  • Automated dispute resolution and legal case management workflows in Camunda, supported by Spring Boot business logic for document tracking and regulatory compliance.
  • Delivered comprehensive reporting and dashboards by combining Java-based services, Camunda history data, and Spring Boot APIs, enabling monitoring of recovery rates, agent performance, and audit trails.
